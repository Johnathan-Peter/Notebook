    OCS2 代码包文件结构解析 ¶

    OCS2 (Optimal Control for Switched Systems) 是一个用于非线性最优控制问题的 C++ 工具箱，专注于机器人的实时模型预测控制 (MPC)。下面是该代码包的详细架构解析：

    核心模块 ¶

    模块名称 功能描述
    ocs2_core 核心数据类型、数学工具、前向积分接口、自动微分 (AD) 工具。是整个框架的基础设施层。
    ocs2_oc 最优控制问题的构建模块，包括：代价函数、约束、系统动力学、预计算、参考轨迹管理等。

    求解器模块 ¶

    模块名称 功能描述
    ocs2_ddp DDP 系列求解器，包含连续时间约束 DDP(SLQ) 和离散时间约束 iLQR 算法。
    ocs2_mpc MPC 基础设施和运行时接口，将求解器封装为 MPC 控制器。
    ocs2_sqp 多重打靶 SQP 求解器，QP 子问题通过 HPIPM/BLASFEO 求解。
    ocs2_slp 序列线性规划求解器，使用 PIPG 算法。
    ocs2_ipm 多重打靶非线性内点法求解器。
    ocs2_frank_wolfe Frank-Wolfe 算法求解器，用于特定优化问题。

    机器人工具模块 ¶

    模块名称 功能描述
    ocs2_pinocchio 基于 Pinocchio 库的机器人建模工具集，包含多个子模块：
    └─ ocs2_pinocchio_interface Pinocchio 接口封装
    └─ ocs2_centroidal_model 质心动力学模型
    └─ ocs2_self_collision 自碰撞检测与约束
    └─ ocs2_sphere_approximation 球体近似碰撞检测
    └─ ocs2_self_collision_visualization 自碰撞可视化工具
    ocs2_robotic_tools 机器人通用工具集，如运动学工具等。
    ocs2_perceptive 感知相关功能模块，处理环境感知与状态估计。
    ocs2_raisim 与 RaiSim 仿真器的集成接口。

    ROS 集成模块 ¶

    模块名称 功能描述
    ocs2_ros_interfaces ROS 接口实现，包括节点、可视化、绘图工具等。
    ocs2_msgs ROS 消息定义文件，定义 MPC 相关的消息类型和服务。

    机器人示例 ¶

    这是最丰富的模块，包含多个完整的最优控制示例：

    示例名称 描述
    ocs2_double_integrator 双积分器系统（最简单的示例，入门推荐）
    ocs2_cartpole 倒立摆系统
    ocs2_ballbot 球形移动机器人
    ocs2_quadrotor 四旋翼无人机
    ocs2_mobile_manipulator 移动机械臂
    ocs2_legged_robot 足式机器人（四足 / 双足）
    ocs2_perceptive_anymal 带感知功能的 ANYmal 四足机器人（最复杂示例）

    每个示例通常包含两个包：

    • ocs2_xxx: 核心算法实现（无 ROS 依赖）
    • ocs2_xxx_ros: ROS 节点封装

    高级功能模块 ¶

    模块名称 功能描述
    ocs2_mpcnet MPC-Net 工具，用于学习策略的部署和训练，结合深度学习。
    ocs2_python_interface Python 绑定接口，支持 Python 调用部分功能。
    ocs2_ocs2 OCS2 元包，用于统一构建所有模块。

    辅助模块 ¶

    模块名称 功能描述
    ocs2_test_tools 测试工具集，用于单元测试和验证。
    ocs2_thirdparty 第三方依赖库的头文件（如 Boost、Eigen 组件等）。
    ocs2_doc 文档源码（使用 Sphinx/Doxygen 生成）。

    学习建议 ¶

    如果你是初学者，建议按以下顺序学习：

    1. 从示例入手：先看 ocs2_double_integrator（最简单）理解基本框架
    2. 理解核心概念：研究 ocs2_core 和 ocs2_oc 的接口设计
    3. 深入求解器：根据需求学习 ocs2_ddp 或其他求解器
    4. 机器人应用：研究 ocs2_pinocchio 和具体的机器人示例
    5. ROS 部署：学习 ocs2_ros_interfaces 进行实际部署

    官方文档地址：https://leggedrobotics.github.io/ocs2/
